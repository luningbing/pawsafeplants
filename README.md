# Free Ecommerce Campaign URL Builder

Static browser-only campaign URL builder for ecommerce sellers.

## Deploy

Deploy this folder as a static site on Vercel.

- Build command: leave empty
- Output directory: leave empty or use default
- Database: not required
- Supabase: not required

## Privacy

The tool runs in the browser. It does not upload generated URLs, campaign names, or GA4 checklist data to a server.
