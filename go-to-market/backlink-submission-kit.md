## Community Post Draft

Title:

Post:
I kept seeing small ecommerce sellers run paid ads without a clean way to check whether source, campaign, and creative labels show up in GA4.

So I made a free no-login tool: https://whichadgotsale.com/

It does three simple things:

- Builds a clean final landing page URL for ad platforms.
- Checks an existing URL for missing or duplicate tracking parameters.
- Gives a GA4 Realtime checklist before launching traffic.

It is meant for beginner operators, not advanced analytics teams. The default output uses readable labels. Advanced macro formats are optional.

Feedback is welcome, especially from people running Meta, TikTok, Google, affiliate, or AppLovin traffic to an ecommerce store.
