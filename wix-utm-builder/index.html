How to build a Wix UTM link

  1. Copy the exact public Wix page URL you will use in the ad or partner post.
  2. Paste it into the homepage builder.
  3. Select the channel and use a readable campaign name.
  4. Copy the generated URL and open it in a fresh browser tab.
  5. Check GA4 Realtime before you put the link into the ad platform.

Common Wix mistakes

  • Using a preview or editor URL instead of the public page URL.
  • Putting the link into a shortener before checking the full URL once.
  • Changing source and medium names every time a campaign launches.
  • Assuming GA4 ecommerce events work before a simple click test is visible in Realtime.
