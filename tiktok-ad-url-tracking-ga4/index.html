Starter TikTok UTM pattern

This pattern is readable enough for a beginner and still useful in GA4 campaign reports.

https://example.com/products/sample-product?utm_source=tiktok&utm_medium=paid_social&utm_campaign=summer_sale_travel_bag&utm_content=spark_ad_ugc_01
  1. Use the real product, collection, or landing page URL.
  2. Keep utm_source stable as tiktok.
  3. Use paid_social for paid TikTok campaigns and a separate medium for creators or organic links.
  4. Use utm_content for creative, Spark Ad, creator, or video angle labels.

Where TikTok tracking usually breaks

  • The URL works in a browser, but a redirect or app link strips the UTMs.
  • The same campaign uses several source or medium names, making GA4 reports hard to read.
  • A creator link, affiliate link, or landing page builder adds its own parameters after your UTMs.
  • TikTok placeholders are copied from another account without verifying that they expand correctly.
  • GA4 is installed, but Realtime is checked too late or from the wrong property.

Static values vs dynamic placeholders

Field Beginner static value Advanced use
utm_campaign summer_sale_travel_bag Campaign name or ID placeholder after account verification.
utm_term prospecting_broad Audience, ad group, placement, or keyword-like detail.
utm_content spark_ad_ugc_01 Ad, video, creator, creative ID, or placement token.

GA4 Realtime test

  1. Open GA4 Realtime before clicking the TikTok landing page URL.
  2. Click the generated link from a browser, not only inside a dashboard preview.
  3. Confirm the page loads and the address bar still contains UTMs.
  4. Check source, medium, campaign, page path, and event count.
  5. Save the working URL pattern before duplicating ads.

Beginner rule

One test click first. If GA4 cannot see one static TikTok click, do not add dynamic placeholders yet.
